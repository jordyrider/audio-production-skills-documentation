    **## Expression Method**
    Audio tags inline in text. Syntax: `[tag]` immediately before the word or phrase it modifies. Layer multiple tags for combined effect: `[hesitant][nervous] I... I didn't mean to say that.`

    No SSML break tags — use `[pause]`, `[short pause]`, `[long pause]` instead.

    Text structure guidance (same as v2 but tags add precision):
    - Natural speech patterns over formal prose
    - Proper punctuation for micro-pacing
    - CAPS for emphasis
    - Ellipses for trailing/uncertainty
    - Tags + text cues together: `[exhausted] I just... can't do this anymore.`

    **## Audio Tags**
    Organise as a table per category:

    Emotions: `[excited]`, `[nervous]`, `[frustrated]`, `[sorrowful]`, `[calm]`, `[curious]`, `[crying]`, `[mischievously]`
    Reactions: `[sigh]`, `[laughs]`, `[gulps]`, `[gasps]`, `[whispers]`, `[laughs harder]`, `[big laugh]`, `[laughs softly]`
    Cognitive beats: `[pauses]`, `[hesitates]`, `[stammers]`, `[resigned tone]`
    Delivery: `[rushed]`, `[drawn out]`, `[deadpan]`, `[flatly]`, `[playfully]`, `[cheerfully]`
    Pause: `[pause]`, `[short pause]`, `[long pause]`
    Character: `[sarcastically]`, `[French accent]`, `[Australian accent]`, `[pirate voice]`

    **## Stability Modes**
    | Mode | Stability Value | Character |
    |------|----------------|-----------|
    | Creative | ~0.3 | Most expressive, highest variability, least consistent |
    | Natural | ~0.5 | Balanced — expressive but more consistent |
    | Robust | ~0.75 | Most consistent, least expressive |
    Recommendation for content production: Creative or Natural.

    **## Non-Determinism**
    v3 is non-deterministic — same prompt + same settings = different output each run. Strategy: generate 2-3 takes, select best. This is by design; use it as a feature for natural variation.
