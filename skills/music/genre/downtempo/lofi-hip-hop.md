## Sonic DNA
- **Tempo:** 70–85 BPM — slow enough to feel unhurried, fast enough to have rhythmic pulse
- **Rhythm:** Loose, slightly swung hip-hop drum pattern — kick on 1 and 3, snare on 2 and 4, shuffled hi-hats
- **Texture:** Dusty, warm, low-fidelity — vinyl crackle and tape saturation are core to the aesthetic
- **Harmony:** Jazzy chord language — maj7, min7, dom9, diminished passing chords; often a two- or four-bar loop
- **Mood:** Nostalgic and introspective — sits in the space between melancholy and comfort
- **Energy:** Minimal dynamics, loop-like, meditative — not building toward a climax

## Instrumentation

| Instrument | Role | Notes |
|------------|------|-------|
| Piano / Rhodes | Lead melodic voice | Electric piano (Rhodes) gives warmth; acoustic piano for crisper texture |
| Acoustic / Clean Electric Guitar | Secondary melody or comping | Light fingerpicked or jazz-comped chords |
| Upright Bass / Electric Bass | Low-end foundation | Warm, round, understated — mostly root movement |
| Sampled / Lo-Fi Drum Kit | Groove | Slightly off-grid feel; kick-snare-hats with subtle shuffle |
| Vinyl Crackle | Texture layer | Constant low-level noise; essential to the aesthetic |
| Saxophone / Flute | Accent melody | Optional; adds jazz flavour — keep sparse |
| Pads / Strings | Atmosphere | Subtle behind the main instruments; provides harmonic bed |
| Rain / Ambient FX | Background texture | Optional; adds depth and coziness |

## Effects

| Effect | Application |
|--------|-------------|
| Vinyl crackle / record noise | Constant background layer — defines the lo-fi aesthetic |
| Low-pass filter | Rolls off high frequencies for that muffled, warm sound |
| Tape saturation | Adds harmonic warmth and gentle distortion to the mix |
| Pitch wobble / tape flutter | Subtle detuning that gives a human, imperfect feel |
| Small room reverb | Short, warm — glues elements together without washing out |
| Bit-crushing (subtle) | Reduces bit depth slightly for textural grit |
| Chorus / modulation on keys | Light shimmer on Rhodes or piano |

## Prompting Patterns

**MiniMax formula:** Style → Mood → Arrangement → Dynamic

**Key rules:**
- Open with `Lofi` as the style label (stable, confirmed in `music-2.5.md`)
- Set BPM between 75–85 — specify explicitly (e.g. `80 BPM`)
- Lead mood with a single evocative word before instruments
- Avoid high-energy descriptors (`energetic`, `driving`, `powerful`) — they break the genre
- Vinyl/dusty texture descriptors reinforce the aesthetic

**Core phrase bank:**

| Phrase / Term | What It Emphasises |
|---------------|-------------------|
| `Lofi, 80 BPM` | Genre + tempo anchor |
| `dusty piano` | Textured, warm piano with lo-fi character |
| `vinyl crackle` | Noise texture, lo-fi aesthetic |
| `jazzy chords` | Maj7/min7 harmonic language |
| `warm Rhodes` | Electric piano warmth |
| `soft sampled drums` | Loose, lo-fi drum kit feel |
| `upright bass` | Acoustic bass warmth |
| `nostalgic` | Core emotional anchor |
| `melancholic` | Bittersweet, introspective mood |
| `cozy` | Warm, comfortable energy |
| `hazy` | Dreamy, diffuse texture |
| `meditative` | Minimal dynamics, loop-like |
| `gentle piano melody` | Spare, unhurried melodic line |
| `rain ambience` | Background texture layer |

**Prompt structure template:**
```
Lofi, [BPM] BPM, [mood word], [lead instrument + descriptor], [secondary instrument], [texture detail], [dynamic descriptor]
```

**Example assembly:**
```
Style: Lofi, 80 BPM
Mood: nostalgic and melancholic
Arrangement: dusty piano melody, warm upright bass, soft sampled drums, vinyl crackle
Dynamic: minimal, loop-like, meditative and unhurried
Combined: Lofi, 80 BPM, nostalgic and melancholic, dusty piano melody, warm upright bass, soft sampled drums, vinyl crackle, minimal and meditative
```
