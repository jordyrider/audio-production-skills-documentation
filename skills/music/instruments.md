# Instruments

## Purpose
Reference index of instruments used in music production prompting. Use this to identify and name instruments accurately when building genre or arrangement descriptions. Cross-referenced by genre docs (instrumentation section) and model docs (which instruments the model reliably produces).

## Notes

- Specificity matters in prompts — `Rhodes electric piano` is more reliable than `piano`, `808 drum machine` is more reliable than `drums`
- Model support varies — consult the relevant model file to see which instruments are reliably produced
- Instrument combinations imply genre — `upright bass + brushed snare + piano` signals jazz without naming it
